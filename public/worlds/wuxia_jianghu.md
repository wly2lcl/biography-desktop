# 武侠江湖 — 天武风云录

## 世界概述

这是一个刀光剑影、快意恩仇的武侠世界。朝廷衰微，江湖势起，六大势力割据争雄。这里有传承千年的武学秘笈、深不可测的隐世高手、暗流涌动的门派纷争。侠、魔、邪、正——界限模糊，选择决定命运。

你，一个初入江湖的无名小卒，将在这片血雨腥风的天地间书写属于自己的传说。

## 地理环境

### 中原
- **应天府** — 天子脚下，朝廷根基。表面繁华，实则各方势力安插眼线
- **洛阳城** — 江湖第一大城，酒楼客栈暗藏情报交易
- **嵩山少林** — 千年古刹，武林泰斗，藏有无上佛法武功

### 江南
- **姑苏** — 温柔富贵乡，慕容世家所在
- **杭州西湖** — 剑冢秘境入口之一
- **铸剑谷** — 天下第一名剑"天问"的诞生地

### 西域
- **昆仑山** — 传说中修仙门派昆仑墟的所在
- **大漠龙门关** — 西域与中原的分界，三不管地带
- **残雪峰** — 魔教总坛所在，终年积雪

### 南疆
- **十万大山** — 毒虫蛊术横行，五毒教盘踞之地
- **洱海月谷** — 隐世高手隐居之地
- **瘴林** — 失传的《百毒真经》藏匿之处

### 东海
- **桃花岛** — 东邪黄氏传承之地，机关阵法遍布
- **蓬莱仙岛** — 传说中有长生不老之术
- **沉船湾** — 海底沉有前朝武库

## 主要势力

### 六合盟（正道之首）
盟主：上官凌云
武学：六合心法 + 天罡剑诀
宗旨：维护江湖秩序，对抗魔教
现状：表面团结，内部派系林立
态度：对玩家友善但需通过考验

### 影月教（魔教）
教主：（神秘莫测，无人知其真容）
武学：影月心经、噬魂大法
宗旨：追求武学极致，不拘正邪
现状：被正道围攻，退守昆仑
态度：对玩家既拉拢又提防

### 铸剑山庄
庄主：欧冶明
专长：天下第一铸剑世家
武学：以剑入道，不修内功只修剑意
地位：中立势力，各派给三分薄面
特殊：若得庄主赏识，可得神兵利器

### 风媒阁
阁主：千面客
专长：情报买卖，天下无事不知
武学：轻功绝世，隐匿无双
特殊：给钱就有消息，从不过问正邪

### 五毒教
教主：蓝凤凰（苗疆圣女）
武学：百毒秘术、蛊术、金蛇剑法
领地：南疆十万大山
态度：人不犯我我不犯人

### 朝廷·六扇门
统领：铁面神捕 慕容追风
职责：缉拿江湖要犯，制衡武林势力
态度：维持平衡，对作乱者格杀勿论
特殊：玩家若选择为朝廷效力，可获得官面身份

## 职业体系

### 剑客
- 入门：基础剑法
- 进阶：天罡剑诀 / 独孤九剑 / 辟邪剑谱
- 极致：无招胜有招，人剑合一
- 侧重：攻击、速度、精准

### 拳师
- 入门：长拳
- 进阶：降龙十八掌 / 太极拳 / 七伤拳
- 极致：内外兼修，一拳开山
- 侧重：力量、防御、内功

### 刺客
- 入门：潜行之术
- 进阶：影月刺杀术 / 无影步
- 极致：来无影去无踪，一击必杀
- 侧重：速度、隐匿、暴击

### 医者
- 入门：基础医理
- 进阶：药王经 / 毒经 / 针灸秘术
- 极致：活死人肉白骨，亦可杀人无形
- 侧重：辅助、解毒、以医入武

### 奇门
- 入门：阵法基础
- 进阶：奇门遁甲 / 机关术 / 符箓术
- 极致：天地为阵，万物为兵
- 侧重：控制、辅助、群战

## 武学体系

武学分为四个境界：
1. **外功** — 招式技巧，勤练可成
2. **内功** — 真气修炼，需名师指点或秘笈
3. **意境** — 天人合一，可遇不可求
4. **道境** — 以武入道，百年一遇

### 传说级武学（得其一可纵横天下）
- 〖天罡剑诀〗— 六合盟镇派之宝
- 〖影月心经〗— 魔教至高秘典
- 〖九阳神功〗— 少林失传绝学
- 〖独孤九剑〗— 破尽天下武功
- 〖长生诀〗— 传说可得长生

## 历史背景

### 三百年前
武学盛世，百家争鸣。天机老人撰《武学纪要》，记载天下武功。

### 两百年前
魔教崛起，正邪大战。少林、武当、峨眉、昆仑、华山、崆峒六派结盟，史称"六合盟"前身。

### 一百年前
魔教教主被六合盟主击败，魔教退隐昆仑。朝廷设六扇门，开始制衡武林。

### 五十年前
铸剑山庄铸成"天问"神剑，天下震动。传说此剑有灵，能择主而侍。

### 当代
魔教蠢蠢欲动，六合盟内部分裂，朝廷暗布棋子。江湖风雨欲来，英雄辈出之时。

## 传说与秘密

### 剑冢的秘密
铸剑山庄后山有一座剑冢，埋藏着历代名剑。传说剑冢最深处藏有一柄"天道之剑"，得之可悟剑道极致。

### 魔教教主之谜
魔教教主从未以真面目示人，有人说是朝廷卧底，有人说是多年前已死的某位大侠，甚至有人说——他/她就在六合盟高层之中。

### 失落的武学总纲
传说中有一部《天武总纲》，记载了天下所有武学的源头。谁能参透，谁就能创造属于自己的武学体系。

### 长生秘闻
蓬莱仙岛并非传说。每六十年，海市蜃楼中会出现通往仙岛的路径。上一次出现是在五十五年前。
