# 🗺️ 地理 — 凡人界（艾瑞瑟拉大陆）

## 一、大陆总览

艾瑞瑟拉大陆呈不规则椭圆形，总面积约 **1200万平方公里**。大陆被三大洋环绕，内陆有七大主要区域。

## 二、七大区域

### 1. 🏰 中央平原
- **面积：** 约300万平方公里
- **地形：** 广袤平原、缓坡丘陵、丰沃农田
- **主要势力：** 奥兰帝国、七王联盟
- **特征：** 人口最密集、文明最发达、魔力浓度中等
- **重要城市：** 帝都「圣辉城」、贸易枢纽「银港」

### 2. 🌲 翡翠森林（精灵领地）
- **面积：** 约150万平方公里
- **地形：** 远古巨木参天的原始森林，灵脉密布
- **主要势力：** 精灵王庭、德鲁伊议会
- **特征：** 魔力浓度极高，外界难以进入，时间流速略慢
- **重要地标：** 「世界之树」尤格德拉希尔、精灵王城「艾尔维林」

### 3. 🏔️ 龙骨荒原（兽人领地）
- **面积：** 约200万平方公里
- **地形：** 赤红荒原、火山群、峡谷、熔岩河
- **特征：** 环境恶劣但矿产资源丰富，魔力浓度不稳定
- **重要地标：** 「万战峡谷」（古战场遗址）、火山「炎魔之喉」

### 4. ❄️ 极北冰原
- **面积：** 约250万平方公里
- **地形：** 万年冰盖、冰川、冻土
- **特征：** 极端寒冷，居住着冰原兽人、雪精灵等稀有种族
- **重要地标：** 「冰封神殿」（远古遗迹）、永冻深渊

### 5. 🌊 西海岸
- **面积：** 约100万平方公里
- **地形：** 海岸线曲折、多港湾、渔业发达
- **主要势力：** 自由贸易城邦联盟
- **特征：** 海上贸易枢纽，冒险者与海盗并存
- **重要城市：** 「翡翠港」、「金锚城」

### 6. ⛈️ 风暴海岸
- **面积：** 约80万平方公里
- **地形：** 悬崖峭壁、暴风雨频发、海流复杂
- **特征：** 被称为"航行者噩梦"，但也出产珍稀海洋材料
- **重要地标：** 「风暴之眼」（天然魔法漩涡）

### 7. 🌅 落日海湾
- **面积：** 约50万平方公里
- **地形：** 半封闭海湾、阳光充沛、沙滩绵延
- **特征：** 旅游胜地，同时是远古沉船遗迹密集区
- **重要地标：** 「沉没之城」亚特兰蒂斯遗址

## 三、特殊地点

| 地点 | 类型 | 描述 |
|------|------|------|
| **万魔法阵** | 远古遗迹 | 上古时代遗留的超大型魔法阵，至今仍有能量流动 |
| **龙之墓地** | 禁地 | 远古巨龙死亡后的埋骨之地，蕴含恐怖龙威 |
| **时空裂隙** | 位面通道 | 偶尔出现的不稳定空间裂缝，通向未知位面 |
| **永恒塔** | 中立区域 | 悬浮于空中的神秘高塔，不属于任何势力，内有无穷知识 |
| **深渊入口** | 危险区域 | 通往冥界的天然通道，亡灵生物的聚集地 |

## 四、海洋

### 三大洋

| 海洋 | 描述 |
|------|------|
| **翡翠海** | 位于大陆中央，精灵领地内海，水质清澈蕴含魔力 |
| **无尽之海** | 大陆南侧广阔海域，深海中有远古海族文明 |
| **暴风洋** | 大陆东侧海域，常年风暴，极少有船只敢于穿越 |

### 海底文明

- **海族王国：** 由人鱼、海妖等种族组成，与陆地种族时有贸易往来
- **远古沉城：** 海底沉睡着多个远古文明遗迹，冒险者的宝库
