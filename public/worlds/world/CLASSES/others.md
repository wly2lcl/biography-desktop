# 📿 其他职业

## ⛪ 牧师 / 神官

> "愿神的光辉照耀你。"

### 概述

牧师是信仰神明的超凡者，通过祈祷获得神明的恩赐，使用神术进行治愈、辅助或战斗。

### 神术分类

| 类型 | 描述 | 举例 |
|------|------|------|
| **治愈神术** | 恢复生命和状态 | 治疗术、复活术、净化术 |
| **增益神术** | 增强自身或队友 | 神圣护盾、力量祝福、速度祝福 |
| **攻击神术** | 对敌人造成伤害 | 神圣之光、审判之锤 |
| **防御神术** | 保护队友 | 神圣结界、守护之壁 |

### 牧师等级

| 阶位 | 描述 |
|------|------|
| **见习牧师** | 初步感知神明 |
| **牧师** | 可使用基础神术 |
| **神官** | 获得神明认可，神术威力提升 |
| **大祭司** | 可沟通神明，获得强大神术 |
| **圣者** | 半神级别，自身即为圣物 |

## 🧪 炼金术师

> "万物皆可炼，真理在坩埚中。"

### 概述

炼金术师是通过炼金术制造药剂、魔法道具和其他物品的职业。他们不擅长战斗，但制造的物品在战斗中发挥关键作用。

### 炼金分类

| 类型 | 描述 | 举例 |
|------|------|------|
| **药剂炼金** | 制造各种药剂 | 生命药水、魔力药水、解毒剂 |
| **附魔炼金** | 为武器/护甲附魔 | 火焰附魔、锋利附魔 |
| **造物炼金** | 制造魔法道具 | 传送卷轴、魔法炸弹 |
| **人体炼金** | 改造人体（禁忌） | 强化药剂、变异药剂 |

### 炼金术师等级

| 阶位 | 描述 |
|------|------|
| **学徒** | 学习基础配方 |
| **炼金术师** | 可制造标准药剂 |
| **高级炼金术师** | 可改良和创造配方 |
| **炼金大师** | 可制造传说级物品 |
| **贤者** | 触及炼金真理，可炼成神器 |

## 📢 召唤师

> "跨越位面的羁绊，回应我的召唤！"

### 概述

召唤师通过与异界生物签订契约，召唤它们来协助战斗。

### 召唤分类

| 类型 | 描述 | 举例 |
|------|------|------|
| **元素召唤** | 召唤元素生物 | 火元素、水精灵 |
| **契约召唤** | 与强大生物签订契约 | 恶魔、天使、巨龙 |
| **亡灵召唤** | 召唤亡灵生物 | 骷髅、幽灵 |
| **随机召唤** | 召唤未知生物 | 命运之门 |

## 🌿 德鲁伊

> "我是森林的守护者，是万物的代言人。"

### 概述

德鲁伊是与自然合一的职业，可变化形态、操控植物、与动物沟通。主要是精灵族的职业。

### 能力

| 能力 | 描述 |
|------|------|
| **变形** | 可化为熊、狼、鹰等动物形态 |
| **植物操控** | 操控藤蔓、树木进行攻击或防御 |
| **自然沟通** | 与动植物沟通，获取信息 |
| **自然治愈** | 利用自然之力进行治疗 |
