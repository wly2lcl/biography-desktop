# 🗡️ 刺客与盗贼体系

> "你永远不会知道我从哪里来，也不会知道我去向何方。"

## 概述

刺客是以速度、隐匿和致命一击为核心的职业。他们不擅长正面交锋，但能在敌人察觉之前结束战斗。

## 刺客属性

| 属性 | 描述 |
|------|------|
| **敏捷** | 核心属性，决定速度和闪避 |
| **暴击率** | 致命一击的概率 |
| **隐匿** | 隐藏身形和气息的能力 |
| **毒素抗性** | 对毒药的抵抗能力 |

## 刺客分类

| 类型 | 特征 | 武器 | 定位 |
|------|------|------|------|
| **影刺客** | 操控暗影，潜行暗杀 | 匕首、暗刃 | 单体爆发 |
| **盗贼** | 开锁、偷窃、陷阱 | 短剑、工具 | 探索/辅助 |
| **毒师** | 用毒高手，持续伤害 | 吹箭、毒刃 | 持续消耗 |
| **赏金猎人** | 追踪、捕猎、生存 | 十字弩、短刀 | 追踪/猎杀 |

## 刺客等级详解

| 阶位 | 特征 | 特殊能力 |
|------|------|----------|
| **见习** | 学习基础潜行 | 可在暗处隐藏 |
| **初级** | 掌握基本暗杀技巧 | 背刺伤害提升 |
| **中级** | 暗影亲和 | 可在中等光照下隐身 |
| **高级** | 暗影操控 | 可操控暗影进行攻击 |
| **大刺客** | 暗影领域 | 展开暗影领域，完全隐身 |
| **暗影大师** | 暗影法则 | 触碰暗影法则，可在影子间瞬移 |
| **无面者** | 传说境界 | 世间无人能识破其真面目 |
