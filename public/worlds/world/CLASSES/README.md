# ⚔️ 职业体系总览

凡人界的超凡之路分为**三大主体系**和**若干副体系**。

## 三大主体系

| 体系 | 核心 | 优势 | 劣势 |
|------|------|------|------|
| **魔法师** | 魔力与元素操控 | 远程攻击、范围伤害、控制能力强 | 身体脆弱、施法需要时间、耗魔大 |
| **战士** | 肉体与意志力 | 近战强悍、防御力强、持续战斗 | 缺乏远程手段、成长慢 |
| **刺客** | 速度与隐匿 | 单体爆发、情报收集、暗杀 | 正面战斗能力弱、容错率低 |

---

## 等级体系（通用）

所有职业遵循统一的实力等级：

### 凡人阶

| 阶位 | 魔法师对应 | 战士对应 | 描述 |
|------|-----------|---------|------|
| **一阶** | 见习魔法师 | 见习战士 | 入门阶段，初步掌握力量 |
| **二阶** | 初级魔法师 | 初级战士 | 可独立完成任务 |
| **三阶** | 中级魔法师 | 中级战士 | 军队中的精锐 |
| **四阶** | 高级魔法师 | 高级战士 | 可担任教官或小队领袖 |
| **五阶** | 大魔法师 | 大战士 | 王国中的顶尖战力 |
| **六阶** | 魔导师 | 战将 | 可影响一场战役的走向 |
| **七阶** | 大魔导师 | 战神 | 凡人界的巅峰存在 |

### 超凡阶

| 阶位 | 描述 |
|------|------|
| **圣域** | 超越凡人极限，掌握一丝法则之力，寿命大幅延长 |
| **半神** | 法则领悟至深，凝聚神力雏形，可开辟神国雏形 |
| **下位神** | 凝聚神格，成为真正的神明 |
| **中位神** | 神格完善，掌控完整法则 |
| **上位神** | 法则领悟至大圆满，众神中的强者 |
| **神王** | 神界最高存在，掌控多条法则 |

---

## 副体系

| 体系 | 描述 |
|------|------|
| **牧师/神官** | 信仰神明，获得神术加持，擅长治愈与辅助 |
| **炼金术师** | 通过炼金术制造药剂、魔法道具，战斗力弱但辅助极强 |
| **召唤师** | 与异界生物签订契约，召唤生物作战 |
| **死灵法师** | 操控亡灵，被视为禁忌之术 |
| **德鲁伊** | 与自然合一，可变化形态，精灵族专属 |
| **符文师** | 刻印符文于武器、铠甲上，赋予特殊能力 |
