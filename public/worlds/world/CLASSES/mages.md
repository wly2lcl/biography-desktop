# 🔮 魔法师体系

> "元素是我的仆从，法则是我的画笔。"

## 概述

魔法师是操控元素力量的超凡者。他们通过冥想感知魔力，学习咒语和手势来引导元素之力，释放各种魔法。

## 魔法师属性

| 属性 | 描述 |
|------|------|
| **魔力值 (MP)** | 施放魔法消耗的资源，通过冥想恢复 |
| **精神力** | 决定可同时操控的魔法数量和复杂度 |
| **元素亲和度** | 对不同元素的天然亲和力，影响学习速度和威力 |

## 魔法分类

### 按属性

| 属性 | 擅长领域 | 代表魔法 |
|------|----------|----------|
| **火系** | 攻击、爆发 | 火球术、烈焰风暴、炎龙吐息 |
| **水系** | 治愈、控制 | 水疗术、冰霜新星、海啸 |
| **风系** | 速度、辅助 | 风之翼、闪电链、龙卷风 |
| **土系** | 防御、地形 | 石墙术、地震、大地之盾 |
| **雷系** | 爆发、麻痹 | 雷击、天罚、雷霆万钧 |
| **光系** | 治愈、净化 | 圣光术、净化之光、神圣裁决 |
| **暗系** | 侵蚀、诅咒 | 暗影箭、腐蚀术、灵魂收割 |

### 按功能

| 类型 | 描述 | 举例 |
|------|------|------|
| **攻击魔法** | 直接对敌人造成伤害 | 火球、冰锥、闪电 |
| **防御魔法** | 保护自己或队友 | 魔法盾、元素护甲 |
| **辅助魔法** | 增强自身或削弱敌人 | 加速术、力量强化 |
| **治愈魔法** | 恢复生命值或状态 | 治疗术、解毒术 |
| **召唤魔法** | 召唤元素生物或异界生物 | 元素召唤、契约召唤 |
| **空间魔法** | 操控空间和维度 | 瞬移、空间切割 |

## 施法要素

| 要素 | 说明 | 是否可以省略 |
|------|------|-------------|
| **咒语** | 引导魔力的语言 | 高阶可省略（默发） |
| **手势** | 引导魔力流动的肢体动作 | 高阶可省略（瞬发） |
| **魔力** | 施法的能量来源 | 不可省略 |
| **精神力** | 控制魔法的精确度 | 不可省略 |
| **施法材料** | 某些强力魔法需要的消耗品 | 视魔法而定 |

## 魔法师等级详解

| 阶位 | 魔力容量 | 可同时操控 | 特殊能力 |
|------|----------|-----------|----------|
| **见习** | 极低 | 1个基础魔法 | 感知魔力 |
| **初级** | 低 | 1-2个低级魔法 | 掌握一种属性的基础魔法 |
| **中级** | 中等 | 2-3个中级魔法 | 可组合魔法 |
| **高级** | 较高 | 3-4个高级魔法 | 可瞬发低级魔法 |
| **大魔法师** | 高 | 多个高级魔法 | 可默发中级魔法 |
| **魔导师** | 极高 | 大量高级魔法 | 可创造新魔法 |
| **大魔导师** | 接近极限 | 近乎无限 | 触及法则，可短暂改变环境 |
