# ⚔️ 战士体系

> "剑之所至，心之所向。"

## 概述

战士是以肉体和意志为核心的超凡者。他们通过艰苦的锻炼强化身体，将意志力转化为战斗力，是近战中的绝对强者。

## 战士属性

| 属性 | 描述 |
|------|------|
| **生命值 (HP)** | 战士的核心生存指标，远超其他职业 |
| **力量** | 决定物理攻击力 |
| **体质** | 决定防御力和恢复力 |
| **敏捷** | 决定速度和闪避 |
| **意志力** | 决定战技威力和持久战斗力 |

## 战士分类

| 类型 | 特征 | 武器 | 定位 |
|------|------|------|------|
| **狂战士** | 力量至上，狂暴攻击 | 双手大剑、巨斧 | 爆发输出 |
| **骑士** | 均衡型，骑乘战斗 | 长枪、单手剑、盾 | 攻防兼备 |
| **剑士** | 技巧型，速度敏捷 | 单手剑、双剑 | 持续输出 |
| **重装战士** | 防御至上，钢铁壁垒 | 巨盾、战锤 | 坦克/前排 |
| **武僧** | 无武器战斗，气功强化 | 拳套、裸手 | 近战控制 |

## 战士等级详解

| 阶位 | 特征 | 特殊能力 |
|------|------|----------|
| **见习** | 基本体能训练 | 可使用基础武器 |
| **初级** | 力量初显 | 掌握基础战技 |
| **中级** | 斗气觉醒 | 可在武器上附着斗气 |
| **高级** | 斗气外放 | 斗气可离体攻击 |
| **大战士** | 斗气实质化 | 斗气可形成护甲或武器 |
| **战将** | 领域展开 | 展开战斗领域，压制周围敌人 |
| **战神** | 战意通神 | 意志与力量合一，可短暂抗衡圣域 |

## 战技分类

| 类型 | 描述 | 举例 |
|------|------|------|
| **攻击战技** | 提升攻击力的技能 | 斩击、冲锋、旋风斩 |
| **防御战技** | 提升防御力的技能 | 铁壁、格挡、反击 |
| **身法战技** | 提升机动性的技能 | 闪避、突进、疾风步 |
| **爆发战技** | 短时间大幅提升战斗力 | 狂暴、燃烧意志、绝地反击 |
| **领域战技** | 高阶战士专属，展开领域 | 剑域、战意领域 |

## 斗气系统

战士的核心力量来源，分为几个阶段：

| 阶段 | 描述 |
|------|------|
| **内息** | 初阶，斗气仅在体内流动 |
| **外放** | 中阶，斗气可离体附着于武器 |
| **实质化** | 高阶，斗气可化为实体形态 |
| **领域** | 巅峰，斗气形成领域，改变周围环境 |
| **融合** | 圣域，斗气与法则融合 |
