# ⛩️ 神明体系

> 神明是高维度的存在，掌控着世界的法则。

## 神明概述

神明是超越了凡人极限的存在，他们凝聚了神格，掌控了完整的法则。神明居住在神界，但他们的影响力延伸到所有位面。

---

## 神界结构

### 神界层次

| 层次 | 描述 | 居住者 |
|------|------|--------|
| **外层神界** | 神界边缘，新晋神明居住地 | 下位神 |
| **中层神界** | 神界核心区域 | 中位神、上位神 |
| **内层神界** | 神界最深处，神王居所 | 神王、创世神（沉睡中） |
| **神国** | 每位神明拥有的独立空间 | 神明及其信徒 |

### 神国

| 属性 | 描述 |
|------|------|
| **定义** | 神明在神界拥有的独立空间 |
| **大小** | 从一座山到一片大陆不等 |
| **规则** | 由神明制定，可自定义物理规则 |
| **入口** | 只有神明允许的生物可进入 |
| **信仰** | 信徒的灵魂可进入神国 |

## 信仰与神术

### 信仰体系

| 要素 | 描述 |
|------|------|
| **信仰值** | 信徒对神明的信仰程度 |
| **祈祷** | 向神明祈求恩赐的方式 |
| **祭祀** | 向神明奉献祭品以获得恩赐 |
| **神迹** | 神明在凡间展现的力量 |

### 神术获取

| 方式 | 描述 |
|------|------|
| **祈祷** | 通过祈祷获得基础神术 |
| **信仰值** | 信仰值越高，可获得更强神术 |
| **神明恩赐** | 神明直接赐予特殊神术 |
| **神物** | 使用与神明相关的神器或圣物 |

### 牧师与神明的关系

| 关系 | 描述 |
|------|------|
| **契约** | 牧师与神明签订契约，获得神术 |
| **忠诚** | 牧师必须忠于神明，违背契约将失去神术 |
| **使命** | 神明会赋予牧师特定的使命 |
| **晋升** | 信仰值达到一定程度可晋升神职 |

## 神战与神明陨落

### 神明陨落

| 要素 | 描述 |
|------|------|
| **原因** | 被其他神明杀死、信仰消失、自我牺牲 |
| **后果** | 神格散落、神国崩塌、信徒失去力量 |
| **复活** | 理论上可能，但需要巨大代价 |

### 死神哈迪斯的陨落

| 项目 | 描述 |
|------|------|
| **时间** | 第二纪元神战期间 |
| **原因** | 被混沌之主重创，神格碎裂 |
| **后果** | 冥界失控，亡灵入侵凡人界 |
| **现状** | 神格碎片散落在冥界各处，等待集齐 |

## 成神之路

凡人如何成为神明：

| 步骤 | 描述 |
|------|------|
| **1. 触碰法则** | 达到大魔导师/战神境界，触碰法则 |
| **2. 领悟法则** | 将法则领悟至大圆满 |
| **3. 凝聚信仰** | 获得大量信徒的信仰 |
| **4. 凝聚神格** | 以法则为核心，信仰为燃料凝聚神格 |
| **5. 神劫** | 度过成神天劫 |
| **6. 飞升** | 飞升至神界，成为下位神 |
