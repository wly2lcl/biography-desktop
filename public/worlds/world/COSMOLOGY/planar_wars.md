# ⚡ 位面战争

> 当位面之间的平衡被打破，战争便不可避免。

## 位面战争概述

位面战争是不同位面之间的冲突，通常由资源争夺、信仰冲突或混沌入侵引发。

---

## 历史上著名的位面战争

### 第一次神战（第二纪元）

| 项目 | 描述 |
|------|------|
| **时间** | 约8000年前 |
| **参战方** | 众神派系A vs 众神派系B |
| **起因** | 信仰之争，对凡人界干预程度的分歧 |
| **结果** | 众神达成协议，不再直接干预凡间 |
| **影响** | 导致世界格局剧变，多个神明陨落 |

### 混沌入侵（第二纪元）

| 项目 | 描述 |
|------|------|
| **时间** | 约7500年前 |
| **参战方** | 众神联军 vs 混沌之主及其仆从 |
| **起因** | 混沌之主趁神战虚弱期入侵 |
| **结果** | 众神合力封印混沌之主 |
| **影响** | 深渊入口形成，混沌生物开始渗透 |

### 第一次位面战争（第四纪元）

| 项目 | 描述 |
|------|------|
| **时间** | 约1200年前 |
| **参战方** | 凡人界联军 vs 冥界亡灵大军 |
| **起因** | 死神陨落导致冥界失控，亡灵入侵 |
| **结果** | 凡人界联军惨胜，亡灵被击退 |
| **影响** | 催生冒险者公会的前身组织 |

## 位面战争要素

### 战争触发条件

| 条件 | 描述 |
|------|------|
| **位面裂缝** | 自然形成的连接通道 |
| **人工开启** | 强大魔法或仪式强行打开 |
| **神明干预** | 神明直接打开位面通道 |
| **混沌渗透** | 混沌能量逐渐侵蚀位面边界 |

### 战争阶段

| 阶段 | 描述 |
|------|------|
| **前兆** | 位面波动、异常现象出现 |
| **渗透** | 敌方先遣部队通过裂缝进入 |
| **全面战争** | 大规模军队入侵 |
| **僵持** | 双方陷入拉锯战 |
| **决战** | 决定战争走向的关键战役 |
| **结局** | 胜利、失败或和平协议 |

## 位面防御

### 防御体系

| 防御 | 描述 |
|------|------|
| **位面结界** | 保护位面的魔法屏障 |
| **守望者** | 专门监视位面波动的组织 |
| **预警系统** | 检测位面裂缝的魔法装置 |
| **快速反应部队** | 随时准备应对入侵的军队 |

### 位面守望者

| 属性 | 描述 |
|------|------|
| **性质** | 中立组织，由多种族组成 |
| **总部** | 永恒塔（据说） |
| **职责** | 监视位面波动、预警入侵、协助防御 |
| **成员** | 圣域及以上强者 |
| **权限** | 可在任何地区活动，不受当地法律限制 |

## 位面旅行

### 旅行方式

| 方式 | 描述 | 风险 |
|------|------|------|
| **自然通道** | 利用现有位面裂缝 | 中等，裂缝可能不稳定 |
| **魔法传送** | 使用空间魔法 | 高，需要极强实力 |
| **神器** | 使用位面相关神器 | 低，但神器稀有 |
| **神明接引** | 由神明带领 | 极低，但需要神明认可 |
| **飞升** | 突破极限后自然飞升 | 无，但门槛极高 |

### 位面旅行注意事项

| 注意事项 | 描述 |
|----------|------|
| **法则差异** | 不同位面法则不同，可能影响实力 |
| **语言障碍** | 不同位面可能有不同语言 |
| **敌意生物** | 可能遇到敌对位面的生物 |
| **环境适应** | 需要适应新位面的环境 |
| **回归困难** | 返回原位面可能比去更难 |
