# 🔥 元素位面

> 纯粹的元素力量在此汇聚，构成了世界的根基。

## 元素位面概述

元素位面是纯粹元素力量的聚集地，分为七个子位面，每个子位面由一位元素领主统治。

### 位面结构

| 子位面 | 元素 | 环境 | 领主 |
|--------|------|------|------|
| **火元素位面** | 火 | 火山、熔岩、火焰平原 | 炎魔领主·伊格尼斯 |
| **水元素位面** | 水 | 海洋、河流、瀑布 | 水精灵女王·艾奎娜 |
| **风元素位面** | 风 | 天空、风暴、气流 | 风之领主·希尔芙 |
| **土元素位面** | 土 | 山脉、平原、洞穴 | 土之巨人·格罗克 |
| **雷元素位面** | 雷 | 雷暴云层、闪电平原 | 雷之领主·托尔 |
| **光元素位面** | 光 | 光明、纯净、神圣 | 光之天使·卢米娜 |
| **暗元素位面** | 暗 | 黑暗、阴影、虚空 | 暗之领主·诺克图娜 |

### 环境特征

| 特征 | 描述 |
|------|------|

| 生物 | 等级 | 描述 |

| 方式 | 描述 | 风险 |

## 元素位面的规则

### 元素平衡

| 规则 | 描述 |
|------|------|
| **相生** | 火生土、土生金、金生水、水生木、木生火 |
| **相克** | 火克金、金克木、木克土、土克水、水克火 |
| **平衡** | 七元素必须保持平衡，否则位面会崩溃 |

### 元素领主会议

| 属性 | 描述 |
|------|------|
| **频率** | 每100年一次 |
| **地点** | 元素位面交汇处 |
| **议题** | 元素平衡、位面安全、与凡人界的关系 |
| **参与者** | 七位元素领主 |
