# 💰 货币与经济体系

## 货币

凡人界通用货币体系，由奥兰帝国主导制定。

### 货币单位

| 单位 | 符号 | 换算 | 描述 |
|------|------|------|------|
| **铜币** | 🟤 1 C | 100 铜币 = 1 银币 | 最基础的货币，用于日常小额交易 |
| **银币** | ⚪ 1 S | 100 银币 = 1 金币 | 中等面额，用于普通商品交易 |
| **金币** | 🟡 1 G | 10 金币 = 1 白金币 | 高价值货币，用于大额交易 |
| **白金币** | ⚪ 1 P | 100 白金币 = 1 钻石币 | 极高价值，用于大型贸易或购买贵重物品 |
| **钻石币** | 💎 1 D | 1000 钻石币 = 1 魔晶币 | 传说级货币，极少流通 |
| **魔晶币** | 🔮 1 M | — | 蕴含魔力的货币，用于魔法交易 |

## 经济体系

### 主要产业

| 产业 | 描述 | 主要地区 |
|------|------|----------|
| **农业** | 粮食、魔法作物种植 | 中央平原 |
| **矿业** | 矿石、魔晶石开采 | 龙骨荒原、迷雾山脉 |
| **渔业** | 海洋捕捞、海产养殖 | 西海岸、落日海湾 |
| **贸易** | 跨区域商品流通 | 西海岸城邦、圣辉城 |
| **魔法工业** | 魔法道具制造、药剂生产 | 星火城、各大魔法学院 |
| **冒险者经济** | 任务奖励、怪物素材交易 | 各地冒险者公会 |

### 贸易路线

| 路线 | 连接地区 | 主要商品 |
|------|----------|----------|
| **中央商路** | 圣辉城 → 西海岸 | 粮食、魔法物品 |
| **北方商路** | 中央平原 → 极北冰原 | 皮毛、冰属性材料 |
| **东方商路** | 中央平原 → 风暴海岸 | 矿产、武器 |
| **翡翠商路** | 翡翠森林 → 各地 | 精灵艺术品、魔法草药（精灵严格控制） |

## 冒险者收入

冒险者是这个世界的重要经济力量。

| 任务等级 | 奖励范围 | 描述 |
|----------|----------|------|
| **E级** | 1-10 银币 | 简单任务：送信、采集 |
| **D级** | 10-50 银币 | 普通任务：讨伐低阶怪物 |
| **C级** | 1-5 金币 | 中等任务：护送、探索 |
| **B级** | 5-50 金币 | 困难任务：讨伐中级怪物 |
| **A级** | 50-500 金币 | 极难任务：讨伐高级怪物、探索遗迹 |
| **S级** | 500-5000 金币 | 传说任务：涉及国家或种族级别 |
| **SS级** | 5000+ 金币 | 史诗任务：可能改变历史走向 |
