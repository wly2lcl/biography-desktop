# 🌌 宇宙观 — 位面结构与世界法则

## 一、位面结构

宇宙由无数个**位面**组成，如同无数气泡漂浮在混沌虚空之中。每个位面拥有自己的法则和物理规则。

### 主要位面

| 位面 | 等级 | 描述 |
|------|------|------|
| **凡人界（艾瑞瑟拉）** | 下位面 | 物质与魔法交汇之地，所有凡俗种族的家园。魔力浓度中等，空间结构稳定。 |
| **冥界** | 中位面 | 亡者的归宿，由死神法则主导。亡灵生物在此繁衍，灵魂在此接受审判或消散。 |
| **元素位面** | 中位面 | 纯粹元素力量的聚集地，分为火元素位面、水元素位面等，元素生物栖居其中。 |
| **神界** | 上位面 | 众神的居所，法则之力最为浓郁。只有神明和极少数凡人可抵达。 |
| **混沌虚空** | — | 位面之间的虚无空间，充满混沌能量，极其危险。位面战争时是主要战场。 |

### 位面通道

- **自然通道：** 某些特殊地点（如空间裂缝、远古祭坛）天然连接不同位面
- **人工通道：** 神级存在或强大的空间魔法可短暂开启通道
- **飞升：** 凡人界强者突破极限后可"飞升"至神界

## 二、世界法则

法则是构成世界运转的**根本规律**，是比魔法更深层的力量。

### 七大基础法则

| 法则 | 属性 | 代表力量 |
|------|------|----------|
| **毁灭法则** | 破坏、消亡 | 最强攻击力，可湮灭万物 |
| **地脉法则** | 物质、大地 | 防御、重力、物质操控 |
| **生命法则** | 生长、治愈 | 恢复、强化生命力 |
| **光暗法则** | 光明/黑暗 | 净化/腐蚀、审判 |
| **空间法则** | 维度、距离 | 瞬移、空间切割、结界 |
| **时间法则** | 流动、因果 | 加速、减速、预知 |
| **灵魂法则** | 意识、精神 | 控制灵魂、读取思想、精神攻击 |

### 法则领悟等级

| 等级 | 名称 | 描述 |
|------|------|------|
| 一阶 | 入门 | 初步感知法则存在，可使用相应属性力量 |
| 二阶 | 领悟 | 理解法则本质，技能威力大幅提升 |
| 三阶 | 掌握 | 将法则融入自身，可自由运用 |
| 四阶 | 融合 | 可融合多种法则，创造独特技能 |
| 五阶 | 大圆满 | 法则领悟至极限，触达神格门槛 |
| 六阶 | 神格凝聚 | 以法则为核心凝聚神格，成就神位 |

## 三、魔力与元素

### 魔力来源

魔力存在于世间万物之中，浓度因地区而异：

- **高魔区：** 远古战场遗迹、灵脉交汇处、精灵圣地
- **中魔区：** 大部分人类王国、魔法学院周边
- **低魔区：** 偏远荒原、魔法荒漠、被封印之地

### 七大元素

| 元素 | 特性 | 对应法则 |
|------|------|----------|
| 🔥 **火** | 狂暴、毁灭、热情 | 毁灭法则的显化 |
| 💧 **水** | 柔和、治愈、变化 | 生命法则的显化 |
| 🌪️ **风** | 迅捷、自由、无形 | 空间法则的侧面 |
| 🌍 **土** | 坚固、沉稳、承载 | 地脉法则的显化 |
| ⚡ **雷** | 极速、威严、审判 | 光暗法则的显化 |
| ☀️ **光** | 神圣、净化、秩序 | 光暗法则（光明面） |
| 🌑 **暗** | 隐秘、吞噬、混沌 | 光暗法则（黑暗面） |
