# 💀 亡灵与恶魔生物

> 死亡不是终点，而是另一种存在的开始。

## 亡灵生物

亡灵是由死者复活或灵魂附着的生物，主要分为自然亡灵和人为亡灵。

### 自然亡灵

| 名称 | 等级 | 危险度 | 描述 | 掉落物 |
|------|------|--------|------|--------|
| **骷髅战士** | 低 | ★★ | 最基本的亡灵生物 | 骨片、灵魂碎片 |
| **幽灵** | 低 | ★★★ | 无形的亡灵，可穿透物理攻击 | 灵魂精华、幽灵粉尘 |
| **僵尸** | 低 | ★★ | 复活的尸体，行动缓慢但力量大 | 腐肉、尸油 |
| **骷髅弓箭手** | 中 | ★★★ | 远程攻击的骷髅 | 骨箭、灵魂碎片 |
| **幽灵骑士** | 中 | ★★★★ | 骑着幽灵马的骑士 | 灵魂核心、幽灵马鞍 |
| **尸巫** | 高 | ★★★★★ | 拥有智慧的亡灵法师 | 命匣、灵魂之书 |

### 人为亡灵

由死灵法师或亡灵教派制造的亡灵。

| 名称 | 等级 | 危险度 | 描述 | 制造方法 |
|------|------|--------|------|----------|
| **骷髅奴仆** | 低 | ★ | 最低级的亡灵仆从 | 基础死灵法术 |
| **缝合怪** | 中 | ★★★ | 多具尸体缝合而成 | 死灵炼金术 |
| **死亡骑士** | 高 | ★★★★★ | 生前是强大战士的亡灵 | 灵魂绑定+死灵法术 |
| **骨龙** | 极高 | ★★★★★★★ | 巨龙的亡灵形态 | 龙骨+强大死灵法术 |
| **巫妖** | 极高 | ★★★★★★★★ | 拥有智慧和魔法的不死法师 | 命匣制作仪式 |

### 亡灵等级

| 等级 | 智力 | 力量 | 特殊能力 |
|------|------|------|----------|
| **低阶** | 无 | 低 | 基本行动 |
| **中阶** | 低 | 中 | 简单指令执行 |
| **高阶** | 中 | 高 | 独立行动，特殊能力 |
| **极高阶** | 高 | 极高 | 拥有生前记忆和智慧 |

### 亡灵弱点

| 弱点 | 描述 |
|------|------|
| **圣光** | 光系魔法对亡灵有额外伤害 |
| **火焰** | 火焰可焚烧亡灵躯体 |
| **灵魂攻击** | 直接攻击灵魂可彻底消灭亡灵 |
| **命匣** | 摧毁命匣可消灭对应的巫妖 |

## 恶魔生物

来自深渊或混沌位面的生物，通常具有邪恶和破坏的本能。

### 深渊恶魔

| 名称 | 等级 | 危险度 | 描述 | 掉落物 |
|------|------|--------|------|--------|
| **小恶魔** | 低 | ★★ | 最低级的恶魔生物 | 恶魔角、恶魔血 |
| **炎魔** | 高 | ★★★★★ | 由火焰构成的恶魔 | 炎魔核心、火焰精华 |
| **梦魇** | 中 | ★★★★ | 操控梦境的恶魔 | 梦魇粉尘、恐惧精华 |
| **魅魔** | 中 | ★★★★ | 诱惑生物的恶魔 | 魅魔之泪、诱惑精华 |
| **深渊领主** | 极高 | ★★★★★★★ | 深渊中的强大存在 | 深渊核心、领主之角 |

### 混沌生物

混沌之主的仆从，存在于混沌虚空中。

| 名称 | 等级 | 危险度 | 描述 | 掉落物 |
|------|------|--------|------|--------|
| **混沌幼虫** | 低 | ★★ | 混沌的基本形态 | 混沌碎片 |
| **混沌兽** | 高 | ★★★★★★★★ | 混沌的具象化 | 混沌精华、混沌结晶 |
| **混沌领主** | 极高 | ★★★★★★★★★ | 混沌位面的统治者 | 混沌核心 |
| **虚空行者** | 中 | ★★★★★ | 在虚空中行走的生物 | 虚空精华 |

### 恶魔等级

| 等级 | 描述 | 对应实力 |
|------|------|----------|
| **低阶恶魔** | 基本的恶魔生物 | 一阶~三阶 |
| **中阶恶魔** | 拥有特殊能力的恶魔 | 四阶~六阶 |
| **高阶恶魔** | 深渊领主级别 | 七阶~圣域 |
| **深渊领主** | 深渊的统治者 | 半神级别 |
| **混沌之主** | 混沌的最高存在 | 神王级别（被封印） |

### 恶魔弱点

| 弱点 | 描述 |
|------|------|
| **圣光** | 光系魔法对恶魔有额外伤害 |
| **神圣武器** | 附魔神圣属性的武器 |
| **封印** | 特定的封印仪式可封印恶魔 |
| **真名** | 知道恶魔真名可控制或驱逐它 |

## 亡灵与恶魔的威胁

### 亡灵潮汐

| 属性 | 描述 |
|------|------|
| **触发** | 死神神格碎片被激活或封印松动 |
| **规模** | 数百万亡灵同时出现 |
| **影响** | 大面积土地被亡灵占据 |
| **应对** | 牧师、圣骑士、光系魔法师 |

### 恶魔入侵

| 属性 | 描述 |
|------|------|
| **触发** | 深渊封印松动或人为召唤 |
| **规模** | 从少量恶魔到恶魔大军 |
| **影响** | 土地被腐蚀，生物变异 |
| **应对** | 封印师、圣骑士、位面守望者 |
