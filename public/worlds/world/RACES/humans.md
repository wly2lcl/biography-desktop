# 🧑 人族 — 好奇之子

> "我们的生命短暂，正因如此，每一刻都要活得精彩。"

## 概述

人族是凡人界数量最多、分布最广的种族。虽然寿命短暂、初始天赋平庸，但凭借极强的**适应性**和**学习能力**，人族建立了最庞大的帝国和最繁荣的文明。

## 生理特征

- **寿命：** 70-90年（修炼者可延长至数百年）
- **身高：** 160-185cm
- **肤色：** 多样，从浅白到深棕皆有
- **魔力亲和：** 中等，但可通过后天学习大幅提升
- **体质：** 中等，无明显强项或弱点

## 种族天赋

| 天赋 | 效果 |
|------|------|
| **快速学习** | 人族学习新技能的速度是其他种族的2-3倍 |
| **意志坚韧** | 面对逆境时，人族突破瓶颈的概率更高 |
| **兼容并蓄** | 可同时修炼多种体系（如魔法+战士），且兼容性优于其他种族 |
| **创造力** | 人族发明新魔法、新武器的概率远高于其他种族 |

## 社会结构

人族建立了多种政治体制：

- **帝国：** 奥兰帝国（中央平原），由皇帝统治，等级森严但秩序井然
- **王国联盟：** 七王联盟，七个独立王国组成的松散联盟
- **城邦：** 西海岸自由贸易城邦，商人治国
- **共和国：** 少数城邦实行议会制

## 文化特色

- **重视教育：** 魔法学院、武技学院遍布人族领地
- **商业发达：** 人族建立了最完善的贸易网络
- **多元信仰：** 崇拜多位神明，没有统一的宗教
- **冒险精神：** 大量人族成为冒险者，探索未知
