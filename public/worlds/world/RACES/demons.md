# 😈 魔族 — 忧虑之子

> "力量不是用来炫耀的，而是用来保护你想保护的东西。"

## 概述

魔族是创世神忧虑情绪所化的种族，天生与**黑暗和灵魂**紧密相连。他们外表可能令人畏惧，但内心往往比任何种族都更懂得珍惜和保护。

## 生理特征

- **寿命：** 500-1000年
- **身高：** 170-200cm
- **特征：** 深色皮肤、尖角（部分）、红瞳或紫瞳、尾巴（部分亚种）
- **魔力亲和：** 极高，尤其对暗系和灵魂魔法
- **体质：** 较强，但不如兽人和矮人

## 魔族分类

| 分类 | 特征 | 聚居地 |
|------|------|--------|
| **纯血魔族** | 最古老的魔族血脉，力量强大 | 魔族王国深处 |
| **混血魔族** | 与其他种族混血，特征不明显 | 各地都有分布 |
| **堕天使** | 曾是天使，后堕落为魔 | 未知 |
| **影魔** | 完全暗影化的魔族 | 暗影位面 |

## 种族天赋

| 天赋 | 效果 |
|------|------|
| **黑暗魔法** | 天生精通暗系魔法，学习速度极快 |
| **灵魂感知** | 可感知灵魂的状态和情绪 |
| **契约能力** | 可与其他生物签订契约，共享力量 |
| **夜视** | 在黑暗中视觉清晰 |
| **恶魔形态** | 部分纯血魔族可变化为恶魔形态，大幅提升战斗力 |

## 社会结构

- **贵族制：** 魔族社会由纯血贵族统治，等级森严
- **魔皇：** 魔族最高统治者，拥有最强血脉
- **议会：** 由各大贵族家族代表组成，商议国家事务

## 文化特色

- **重视荣誉：** 魔族极其重视个人荣誉和家族名誉
- **契约精神：** 魔族签订的契约具有魔法约束力，违约者会受到反噬
- **复杂情感：** 外表冷漠但内心情感丰富，一旦认可某人会极其忠诚
- **艺术天赋：** 魔族在音乐、绘画等艺术领域有极高造诣

## 魔族王国

| 区域 | 描述 |
|------|------|
| **暗影王城** | 魔族首都，建在巨大的暗影水晶之上 |
| **契约之城** | 专门处理契约事务的城市，中立地区 |
| **深渊边缘** | 靠近深渊的危险区域，魔族前哨站 |

## 重要人物

| 姓名 | 身份 | 事迹 |
|------|------|------|
| **路西法·暗影** | 魔皇 | 统一魔族各部，建立强大王国 |
| **莉莉丝·夜歌** | 外交官 | 促成魔族与人族的和平协议 |
| **该隐·契约** | 契约大师 | 制定魔族契约法典，沿用至今 |
