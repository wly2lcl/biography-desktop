# 🐺 兽人族 — 愤怒之子

> "强者生存，弱者被淘汰——这就是荒原的法则。"

## 概述

兽人族是创世神愤怒情绪所化的种族，天生体魄强健、崇尚武力。他们居住在环境恶劣的龙骨荒原，在残酷的环境中锤炼出钢铁般的意志。

## 生理特征

- **寿命：** 60-80年（修炼者可延长）
- **身高：** 190-220cm，部分变异兽人可达250cm
- **特征：** 獠牙、粗犷面容、不同亚种有兽类特征（狼耳、牛尾等）
- **魔力亲和：** 极低，几乎无法学习常规魔法
- **体质：** 极强，恢复力和耐力远超其他种族

## 兽人亚种

| 亚种 | 特征 | 特殊能力 |
|------|------|----------|
| **狼兽人** | 狼耳狼尾，敏捷型 | 群体狩猎时战斗力倍增 |
| **牛兽人** | 牛角、体型巨大 | 力量和防御最强 |
| **虎兽人** | 虎纹、独来独往 | 单体战斗力最强，潜行能力 |
| **鹰兽人** | 羽翼、视力极佳 | 可在空中滑翔，侦察能力 |
| **蛇兽人** | 蛇鳞、冷血 | 毒素抗性，夜视能力 |

## 种族天赋

| 天赋 | 效果 |
|------|------|
| **狂暴** | 进入狂暴状态后，力量和速度提升50%，持续时间内感受不到疼痛 |
| **肉体恢复** | 伤口愈合速度是其他种族的3-5倍 |
| **战吼** | 发出战吼可提升周围兽人士气，威慑敌人 |
| **环境适应** | 对极端温度和恶劣环境有极强耐受力 |

## 社会结构

- **部落制：** 兽人以部落为基本单位，每个部落由最强者（酋长）统治
- **万战议会：** 各部落酋长组成的议事机构，每十年在万战峡谷召开
- **力量至上：** 无论种族、年龄、性别，只有强者才能获得尊重

## 文化特色

- **武力崇拜：** 力量是衡量一切的标准
- **荣誉文化：** 战死沙场是最高荣誉，怯懦是最深的耻辱
- **口述传承：** 没有文字，历史和知识通过歌谣和故事代代相传
- **战舞仪式：** 重大事件通过战舞庆祝或祈祷
