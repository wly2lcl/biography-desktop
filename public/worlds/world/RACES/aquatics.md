# 🧜 海族 — 生命女神的子民

> "海洋是母亲的怀抱，每一滴水都有记忆。"

## 概述

海族是由生命女神艾莉娅独立创造的种族，生活在海洋深处。他们与陆地种族有贸易往来，但很少离开海洋。

## 生理特征

- **寿命：** 200-400年
- **身高：** 150-170cm（陆地形态）
- **特征：** 鳃（颈部两侧）、鳞片（部分身体）、蹼状手指和脚趾
- **魔力亲和：** 高，尤其对水系魔法
- **体质：** 中等，水下远强于陆地

## 海族分类

| 分类 | 特征 | 聚居地 |
|------|------|--------|
| **人鱼族** | 上半身类人，下半身鱼尾 | 珊瑚城 |
| **海妖族** | 容貌绝美，歌声有魔力 | 深海宫殿 |
| **海妖族** | 鱼头人身，力量强大 | 深海洞穴 |
| **深海族** | 适应深海高压环境 | 深渊边缘 |

## 种族天赋

| 天赋 | 效果 |
|------|------|
| **水下呼吸** | 可在水下自由呼吸 |
| **水压适应** | 可适应深海高压环境 |
| **水语者** | 可与海洋生物沟通 |
| **歌声魔力** | 海妖族的歌声可影响他人心智 |
| **水流操控** | 可操控水流进行攻击或防御 |

## 社会结构

- **海王国：** 海族的统一王国，由海王统治
- **珊瑚议会：** 由各海族代表组成，商议国家事务
- **深海长老：** 最年长的海族，拥有智慧和建议权

## 文化特色

- **海洋崇拜：** 将海洋视为母亲，极度尊重和保护海洋
- **歌声传统：** 通过歌声传递历史和知识
- **珍珠文化：** 珍珠是海族的货币和装饰
- **陆地贸易：** 与陆地种族有贸易往来，但限制严格

## 海王国

| 区域 | 描述 |
|------|------|
| **珊瑚城** | 海王国首都，由巨大珊瑚建造 |
| **珍珠宫** | 海王居住地，装饰着无数珍珠 |
| **沉船湾** | 海族与陆地种族的贸易港口 |
