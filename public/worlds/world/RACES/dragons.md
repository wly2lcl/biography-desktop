# 🐉 龙族 — 威严之子

> "凡人，你是在与时间本身对话。"

## 概述

龙族是创世神威严情绪所化的种族，凡人界最古老、最强大的存在。他们天生拥有巨龙形态，可化为人形，精通龙语魔法，寿命长达数千年。

## 生理特征

- **寿命：** 3000年以上，理论上可永生（除非被杀死）
- **体型：** 巨龙形态体长15-50米不等，化为人形后170-200cm
- **特征：** 龙鳞、龙翼、龙角（巨龙形态）；人形时保留竖瞳和部分龙类特征
- **魔力亲和：** 极高，龙语魔法是世间最古老的魔法体系
- **体质：** 极强，龙鳞可抵御大部分物理和魔法攻击

## 龙族等级

| 等级 | 年龄 | 体长 | 实力 |
|------|------|------|------|
| **幼龙** | 0-100年 | 3-8米 | 相当于初级魔法师 |
| **青年龙** | 100-500年 | 8-20米 | 相当于高级魔法师 |
| **成年龙** | 500-1500年 | 20-35米 | 相当于大魔导师 |
| **远古龙** | 1500-3000年 | 35-50米 | 相当于圣域强者 |
| **古龙** | 3000年以上 | 50米+ | 半神级别，可化为人形 |
| **龙神** | ？ | 未知 | 传说存在，掌控法则 |

## 龙语魔法

龙语魔法是龙族独有的魔法体系，使用**龙语**（世间最古老的语言）直接操控法则：

- **无需咒语/手势：** 龙语本身就是力量
- **威力极大：** 同等级的龙语魔法远超人类魔法
- **学习门槛：** 非龙族极难学会龙语，且需要龙族的认可

## 种族天赋

| 天赋 | 效果 |
|------|------|
| **龙语魔法** | 使用龙语施放魔法，威力远超常规魔法 |
| **龙威** | 低等级生物面对巨龙会因恐惧而丧失战意 |
| **变化形态** | 可在巨龙和人形之间自由切换 |
| **元素抗性** | 对不同元素的抗性取决于龙的属性 |
| **灵魂强大** | 灵魂强度是同等实力其他种族的数倍 |

## 龙族分类

| 类型 | 元素属性 | 特征 |
|------|----------|------|
| **红龙** | 火 | 最暴躁好战，栖息于火山地带 |
| **蓝龙** | 雷 | 最智慧，喜欢研究魔法 |
| **绿龙** | 风 | 最善交际，常与人族往来 |
| **白龙** | 冰 | 栖息于极北冰原 |
| **黑龙** | 暗 | 最神秘，行踪不定 |
| **金龙** | 光 | 最稀有，传说与神明有关 |

## 龙族文化

- **高傲：** 认为自己是世间最完美的种族
- **收藏癖：** 喜欢收集宝物、知识、艺术品
- **契约精神：** 一旦许下诺言，必遵守到底
- **隐居：** 大部分龙族喜欢独处，不喜群居
