# ⚒️ 矮人族 — 坚忍之子

> "山不会移动，但我们会把山变成我们想要的样子。"

## 概述

矮人族是创世神坚忍情绪所化的种族，天生与**大地和火焰**紧密相连。他们是世间最出色的工匠和锻造师，居住在山脉深处，建造了令人惊叹的地下城邦。

## 生理特征

- **寿命：** 300-400年
- **身高：** 120-140cm
- **特征：** 粗壮体型、浓密胡须（男女皆有）、宽厚手掌
- **魔力亲和：** 较低，但对土系和火系魔法有特殊亲和
- **体质：** 极强，尤其是对物理伤害和毒素的抵抗力

## 矮人分类

| 分类 | 特征 | 聚居地 |
|------|------|--------|
| **山地矮人** | 最传统的矮人，擅长锻造 | 迷雾山脉地下城邦 |
| **丘陵矮人** | 体型略小，擅长采矿 | 丘陵地带 |
| **灰矮人** | 皮肤呈灰色，适应地下生活 | 深层地下洞穴 |
| **火矮人** | 对火焰有特殊亲和 | 火山地带 |

## 种族天赋

| 天赋 | 效果 |
|------|------|
| **锻造大师** | 制造武器和护甲的品质天然高于其他种族 |
| **毒素免疫** | 对所有毒素和疾病免疫 |
| **黑暗视觉** | 在完全黑暗中也能看清30米范围内的事物 |
| **岩石亲和** | 对岩石和矿物有天然的感知能力 |
| **坚韧不拔** | 面对逆境时不会轻易放弃，意志力极强 |

## 社会结构

- **氏族制：** 矮人以氏族为基本单位，每个氏族有独特的锻造传统
- **工匠议会：** 由各氏族最年长、最杰出的工匠组成，决定重大事务
- **国王：** 矮人王由工匠议会推选，通常是技艺最高超的工匠

## 文化特色

- **锻造崇拜：** 将锻造视为神圣的艺术，每一件作品都倾注心血
- **荣誉文化：** 以作品质量衡量一个人的价值
- **契约精神：** 一旦承诺，必遵守到底，违约者会被全族唾弃
- **饮酒文化：** 矮人酿制的麦酒和烈酒闻名于世

## 著名城邦

| 城邦 | 描述 | 特色 |
|------|------|------|
| **铁炉堡** | 最大的矮人城邦，位于迷雾山脉深处 | 拥有世间最大的熔炉 |
| **金锤城** | 商业最发达的矮人城邦 | 与人类贸易往来密切 |
| **炎之心** | 建在活火山内部的城邦 | 火矮人的圣地 |
