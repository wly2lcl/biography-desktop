## 种族起源

### 创世传说

远古时期，**创世神阿鲁恩**以自身神力创造了世界，并用自己的七种情绪化作了七个初始种族：

- **喜悦** → 精灵
- **愤怒** → 兽人
- **好奇** → 人族
- **坚忍** → 矮人
- **温柔** → 半身人
- **威严** → 龙族
- **忧虑** → 魔族

海族则是后来由生命女神独立创造的种族。
