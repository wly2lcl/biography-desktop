# ⚔️ 武器

## 武器分类

| 类型 | 描述 | 适用职业 |
|------|------|----------|
| **单手剑** | 灵活、平衡 | 战士、骑士 |
| **双手剑** | 高攻击、慢速度 | 狂战士 |
| **法杖** | 增幅魔法威力 | 魔法师、牧师 |
| **弓** | 远程攻击 | 游侠、精灵 |
| **匕首** | 快速、隐匿 | 刺客、盗贼 |
| **斧** | 高破坏力 | 狂战士、兽人 |
| **锤** | 破甲、高冲击 | 重装战士 |
| **长枪** | 骑乘战斗 | 骑士 |
