# 📦 物品分级总览

凡人界的物品按品质和稀有度分为多个等级。

## 物品等级

| 等级 | 颜色标识 | 描述 | 获取途径 |
|------|----------|------|----------|
| **凡铁** | ⚪ 白色 | 普通物品，无特殊属性 | 铁匠铺、商店 |
| **精良** | 🟢 绿色 | 品质优良，有少量加成 | 优秀工匠、任务奖励 |
| **稀有** | 🔵 蓝色 | 附有魔法或特殊效果 | 冒险者公会、副本 |
| **史诗** | 🟣 紫色 | 拥有强大力量和独特属性 | 远古遗迹、击败强敌 |
| **传说** | 🟠 橙色 | 世间罕见，拥有传奇故事 | 历史名人遗物、特殊事件 |
| **神器** | 🔴 红色 | 神明级别的力量 | 神明赐予或锻造 |
| **创世级** | ⭐ 金色 | 创世神级别的力量 | 仅理论存在 |

## 物品分类

| 分类 | 描述 |
|------|------|
| **武器** | 剑、斧、弓、法杖、匕首等 |
| **护甲** | 盔甲、盾牌、长袍、皮甲等 |
| **饰品** | 戒指、项链、耳环、护符等 |
| **消耗品** | 药水、卷轴、食物等 |
| **材料** | 矿石、草药、怪物掉落物等 |
| **奇物** | 拥有特殊能力的稀有物品 |

## 物品属性

| 属性 | 描述 |
|------|------|
| **基础属性** | 攻击力、防御力、耐久度等 |
| **附加属性** | 额外的属性加成（力量、敏捷等） |
| **特殊效果** | 独特的能力或触发效果 |
| **套装效果** | 多件同套装物品装备时的额外加成 |
| **成长性** | 部分物品可通过使用或材料升级 |
