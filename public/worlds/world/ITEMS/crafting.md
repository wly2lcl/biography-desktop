# 🔨 锻造与附魔系统

> 一件好武器，需要匠人的心血和材料的灵魂。

## 锻造体系

### 锻造等级

| 等级 | 名称 | 描述 | 可锻造物品等级 |
|------|------|------|----------------|
| 一阶 | 学徒 | 学习基础锻造技术 | 凡铁 |
| 二阶 | 铁匠 | 可独立锻造 | 凡铁~精良 |
| 三阶 | 高级铁匠 | 技术熟练，可锻造附魔武器 | 精良~稀有 |
| 四阶 | 锻造师 | 掌握特殊锻造技术 | 稀有~史诗 |
| 五阶 | 锻造大师 | 可锻造传说级物品 | 史诗~传说 |
| 六阶 | 锻造宗师 | 世间最顶尖的锻造师 | 传说~神器 |
| 七阶 | 神级锻造师 | 传说存在，可锻造神器 | 神器 |

### 锻造流程

| 步骤 | 描述 |
|------|------|
| **选材** | 选择合适的矿石和材料 |
| **熔炼** | 将材料熔化，去除杂质 |
| **锻造** | 反复锤打，塑造形状 |
| **淬火** | 用特殊液体冷却，增强硬度 |
| **打磨** | 精细打磨，提升品质 |
| **附魔** | （可选）为武器附加魔法效果 |
| **开锋** | 最后一步，赋予武器"灵魂" |

### 锻造材料

| 材料 | 等级 | 描述 | 来源 |
|------|------|------|------|
| **铁矿石** | 凡级 | 最基础的锻造材料 | 矿山 |
| **精钢** | 精良 | 优质钢材 | 精炼铁矿石 |
| **秘银** | 稀有 | 轻便且坚固的魔法金属 | 深层矿山 |
| **精金** | 史诗 | 极其坚固的金属 | 稀有矿山、怪物掉落 |
| **龙钢** | 传说 | 掺入龙血的钢材 | 龙骨、龙血、特殊熔炼 |
| **星辰铁** | 神级 | 从天而降的神秘金属 | 陨石 |

## 附魔体系

### 附魔分类

| 类型 | 描述 | 举例 |
|------|------|------|
| **属性附魔** | 附加元素属性 | 火焰、冰霜、闪电 |

| **套装附魔** | 多件物品组合效果 | 套装加成 |

| **神级** | 神级附魔 | 10% |

## 炼金术

### 炼金分类

| 类型 | 描述 | 产品 |
|------|------|------|
| **药剂炼金** | 制造各种药剂 | 生命药水、魔力药水、属性药剂 |
| **附魔炼金** | 为物品附魔 | 附魔武器、附魔护甲 |
| **造物炼金** | 制造魔法道具 | 传送卷轴、魔法炸弹、储物戒指 |
