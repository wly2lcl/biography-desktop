# 🔱 神器

> 神明级别的物品，拥有改变世界的力量。

## 神器概述

神器是神明锻造或赐福的物品，拥有**法则级别的力量**。每一件神器都与一位或多位神明相关，普通人无法完全发挥其威力。

---

## 七大创世神器

传说创世神阿鲁恩创造世界时使用的七件物品，蕴含着创世之力。

| 神器名 | 所属神明 | 形态 | 能力 |
|--------|----------|------|------|
| **创世之刃** | 创世神阿鲁恩 | 剑 | 可斩断一切，包括法则和位面 |
| **生命之杖** | 生命女神艾莉娅 | 法杖 | 可创造生命，治愈一切伤痛 |
| **死亡之书** | 死神哈迪斯 | 书 | 可决定谁该死、谁该活 |
| **秩序之冠** | 秩序之神朱庇特 | 皇冠 | 可制定规则，违反者将受到惩罚 |
| **混沌之核** | 混沌之主卡俄斯 | 核心 | 可颠覆一切规则，创造混乱 |
| **时空之轮** | 时空之神 | 轮盘 | 可操控时间流动和空间结构 |
| **元素王座** | 元素之神 | 王座 | 可完全掌控所有元素力量 |

### 创世之刃

| 属性 | 描述 |
|------|------|
| **形态** | 一把散发着金色光芒的长剑 |

| **限制** | 只有被创世神认可者可使用 |

| **传说** | 据说拥有创世之刃者可重塑世界 |

### 生命之杖

| 属性 | 描述 |
|------|------|
| **形态** | 一根由世界之树枝条制成的法杖 |
| **威力** | 可创造生命、治愈一切伤痛、复活死者 |
| **限制** | 只有信仰生命女神者可使用 |
| **下落** | 由生命女神保管于神国 |
| **传说** | 据说用生命之杖点化的土地可长出永不凋零的花朵 |

### 死亡之书

| 属性 | 描述 |
|------|------|
| **形态** | 一本黑色封皮的书，书页上写满了名字 |

| **限制** | 只有死神或死神继承者可使用 |

## 神器的使用

### 使用条件

| 条件 | 描述 |
|------|------|
| **认可** | 神器必须认可使用者才能发挥全部威力 |
| **力量** | 使用者必须达到一定实力等级 |
| **信仰** | 部分神器需要信仰对应神明 |
| **血脉** | 部分神器需要特定血脉才能使用 |

### 神器等级

| 等级 | 描述 | 可使用境界 |
|------|------|------------|
| **凡级神器** | 神明锻造的普通物品 | 大魔导师/战神以上 |
| **次级神器** | 拥有部分法则之力 | 圣域以上 |
| **主级神器** | 拥有完整法则之力 | 半神以上 |
| **创世神器** | 创世级别的物品 | 神明级别 |

### 神器认主

| 方式 | 描述 | 成功率 |
|------|------|--------|
| **血脉认主** | 神器认可使用者的血脉 | 100%（如果血脉匹配） |
| **战斗认主** | 在战斗中证明自己的实力 | 50% |
| **信仰认主** | 通过信仰获得神器认可 | 70%（信仰越深越高） |
| **试炼认主** | 通过神器设置的试炼 | 30% |
| **契约认主** | 与神器签订契约 | 40% |

## 神器的诅咒

部分神器带有诅咒，使用者需要付出代价。

| 神器 | 诅咒 | 描述 |
|------|------|------|
| **死亡之书** | 生命代价 | 每次使用减少寿命 |
| **混沌之核** | 理智丧失 | 使用者逐渐失去理智 |
| **暗影之袍** | 灵魂侵蚀 | 长期使用会被暗影吞噬 |
| **战神之盾** | 战斗狂热 | 使用者会变得好战 |
