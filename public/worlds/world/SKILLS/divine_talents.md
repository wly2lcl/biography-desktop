# ⛪ 神术与天赋技能

> 神明的恩赐与种族的天赋，是超凡之路的重要力量。

## 神术

神术是神明赐予信徒的力量，通过信仰获得。

### 神术来源

| 来源 | 描述 |
|------|------|
| **祈祷** | 通过祈祷获得基础神术 |
| **信仰值** | 信仰值越高，可获得更强神术 |
| **神明恩赐** | 神明直接赐予特殊神术 |
| **神物** | 使用与神明相关的神器或圣物 |

### 治愈神术

| 技能名 | 等级 | 描述 | 效果 |
|--------|------|------|------|
| **治疗术** | 凡级 | 基础治愈魔法 | 恢复少量生命值 |
| **高级治疗术** | 灵级 | 更强的治愈能力 | 恢复中等生命值 |
| **群体治疗术** | 灵级 | 治疗多个目标 | 恢复小队成员生命值 |
| **复活术** | 地级 | 复活死者 | 复活刚死亡的目标 |
| **完全治愈** | 天级 | 治愈一切伤痛 | 恢复全部状态，清除所有负面效果 |
| **生命之泉** | 神级 | 召唤生命泉水 | 完全恢复并净化所有负面状态 |

### 增益神术

| 技能名 | 等级 | 描述 | 效果 |
|--------|------|------|------|
| **力量祝福** | 凡级 | 增强力量 | 力量+20%，持续30分钟 |
| **敏捷祝福** | 凡级 | 增强敏捷 | 敏捷+20%，持续30分钟 |
| **智力祝福** | 灵级 | 增强智力 | 智力+20%，持续30分钟 |
| **神圣护盾** | 灵级 | 神圣保护 | 吸收伤害的护盾 |
| **群体祝福** | 地级 | 增强整个小队 | 全属性+10%，持续1小时 |
| **神之恩赐** | 天级 | 神明的直接恩赐 | 短时间内全属性+50% |

### 攻击神术

| 技能名 | 等级 | 描述 | 效果 |
|--------|------|------|------|
| **神圣之光** | 凡级 | 发射神圣光线 | 对邪恶生物造成额外伤害 |
| **审判之锤** | 灵级 | 召唤神圣之锤 | 范围神圣伤害 |
| **神圣风暴** | 地级 | 召唤神圣风暴 | 大范围神圣伤害，对亡灵/恶魔翻倍 |
| **天罚** | 天级 | 召唤天罚之雷 | 单体极高伤害，对邪恶目标必杀 |
| **神之怒** | 神级 | 神明的愤怒 | 毁灭性范围攻击，仅上位神可用 |

### 防御神术

| 技能名 | 等级 | 描述 | 效果 |
|--------|------|------|------|
| **神圣结界** | 凡级 | 基础防御结界 | 减少受到的伤害 |
| **守护之壁** | 灵级 | 神圣之墙 | 阻挡物理和魔法攻击 |
| **绝对守护** | 地级 | 绝对防御 | 短时间内免疫一切伤害 |
| **神之庇护** | 天级 | 神明的庇护 | 免疫即死效果，恢复生命值 |

### 人族天赋

| 天赋 | 描述 | 触发条件 |
|------|------|----------|
| **快速学习** | 学习新技能的速度是其他种族的2-3倍 | 被动 |
| **意志坚韧** | 面对逆境时，突破瓶颈的概率更高 | 被动 |
| **兼容并蓄** | 可同时修炼多种体系 | 被动 |
| **创造力** | 发明新魔法、新武器的概率更高 | 被动 |

## 特殊天赋技能

### 血脉觉醒

某些种族或个体在特定条件下可觉醒隐藏血脉：

| 血脉 | 描述 | 觉醒条件 |
|------|------|----------|
| **龙之血脉** | 获得部分龙族能力 | 与龙族签订契约或吸收龙血 |
| **神之血脉** | 获得神明赐福 | 完成神明试炼 |
| **远古血脉** | 觉醒远古祖先的力量 | 在远古遗迹中接受传承 |
| **混沌血脉** | 获得混沌之力 | 接触混沌能量 |

### 顿悟

在特定情况下可能触发顿悟：

| 触发条件 | 描述 | 效果 |
|----------|------|------|
| **生死边缘** | 在生死战斗中 | 突然领悟新技能或突破瓶颈 |
| **自然感悟** | 观察自然现象 | 理解法则或创造新魔法 |
| **情感爆发** | 极度愤怒/悲伤/喜悦 | 短时间内力量暴增 |
| **传承接受** | 接受强者传承 | 获得大量知识和技能 |
