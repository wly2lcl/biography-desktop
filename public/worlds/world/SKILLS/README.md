# ✨ 技能体系

> 技能是职业能力的具体体现，是超凡者战斗的工具。

## 技能分类

| 分类 | 描述 | 适用职业 |
|------|------|----------|
| **魔法技能** | 操控元素和法则的技能 | 魔法师、部分其他职业 |
| **武技/战技** | 以肉体和斗气为基础的技能 | 战士、部分其他职业 |
| **神术** | 信仰神明获得的技能 | 牧师、神官 |
| **天赋技能** | 种族或个体独有的技能 | 所有种族 |
| **辅助技能** | 非战斗类技能 | 所有职业 |

## 技能等级

| 等级 | 描述 | 获取难度 |
|------|------|----------|
| **凡级** | 最基础的技能，大部分人都可学习 | 容易 |
| **灵级** | 需要一定天赋和理解力 | 中等 |
| **地级** | 强大的技能，需要较高阶位才能使用 | 困难 |
| **天级** | 传说级技能，可改变战场局势 | 极难 |
| **神级** | 神明级别的技能，触碰法则 | 几乎不可能 |
| **创世级** | 创世神级别的技能 | 仅理论存在 |

## 技能修炼

技能的修炼分为几个阶段：

| 阶段 | 描述 |
|------|------|
| **入门** | 初步学会技能，但威力有限 |
| **熟练** | 可稳定施放，威力正常 |
| **精通** | 技能威力提升，可变化使用方式 |
| **圆满** | 技能修炼至极限，可与其他技能融合 |
| **化境** | 技能融入本能，无需思考即可使用 |
