# 🏛️ 势力与组织

## 王国与帝国

### 🏰 奥兰帝国

- **位置：** 中央平原
- **人口：** 约3000万（人族为主，多种族混居）
- **统治者：** 皇帝「奥兰十七世」
- **军事：** 帝国骑士团、魔法军团、步兵军团
- **特征：** 凡人界最强大的国家，拥有完善的行政体系和强大的军队

### 🌲 精灵王庭

- **位置：** 翡翠森林
- **人口：** 约50万（精灵族为主）
- **统治者：** 精灵王「艾尔瑟·银月」
- **军事：** 精灵弓箭手、德鲁伊、魔法护卫
- **特征：** 封闭但和平，对外交流有限

### 🏔️ 兽人万战联盟

- **位置：** 龙骨荒原
- **人口：** 约200万（兽人族为主）
- **统治者：** 万战议会（七大部落酋长组成）
- **军事：** 部落战士、萨满、战争巨兽
- **特征：** 尚武文化，定期举行万战大会

### 🌊 自由贸易城邦联盟

- **位置：** 西海岸
- **人口：** 约1500万（多种族混居）
- **统治者：** 商人议会
- **军事：** 雇佣兵、海军
- **特征：** 商业发达，冒险者聚集地

## 公会与组织

### ⚔️ 冒险者公会

- **性质：** 中立组织，多种族联合
- **总部：** 圣辉城（奥兰帝国帝都）
- **功能：** 发布任务、评级冒险者、提供情报
- **等级：** 铜牌 → 银牌 → 金牌 → 白金 → 钻石 → 传奇

### 🎓 魔法学院联盟

- **性质：** 教育与研究机构
- **总部：** 星火城
- **功能：** 培养魔法师、研究魔法理论
- **分支：** 七属性魔法学院、战斗魔法学院、理论魔法学院

### 🛡️ 守护骑士团

- **性质：** 帝国军事组织
- **总部：** 圣辉城
- **功能：** 保卫帝国、讨伐邪恶、保护平民
- **等级：** 见习骑士 → 骑士 → 圣骑士 → 守护骑士 → 大骑士

### 🗡️ 暗影兄弟会

- **性质：** 秘密刺客组织
- **总部：** 未知
- **功能：** 暗杀、情报收集、秘密行动
- **特征：** 极度隐秘，据说成员遍布世界各地

### 🌿 德鲁伊议会

- **性质：** 自然守护者
- **总部：** 翡翠森林深处
- **功能：** 维护自然平衡、保护濒危物种
- **成员：** 主要是精灵和部分人类德鲁伊

### 💀 亡灵教派

- **性质：** 禁忌组织
- **总部：** 深渊入口附近
- **功能：** 研究死灵法术、试图复活死者
- **特征：** 被所有正统势力视为敌人
