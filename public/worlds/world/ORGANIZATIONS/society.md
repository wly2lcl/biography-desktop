# 🏠 日常生活与社会制度

> 超凡者的世界很精彩，但普通人的生活同样值得记录。

## 社会阶层

### 人族社会

| 阶层 | 描述 | 占比 |
|------|------|------|
| **贵族** | 皇室、公爵、伯爵等 | 1% |
| **骑士/军官** | 军事阶层 | 3% |
| **商人** | 富有的商人阶层 | 5% |
| **自由民** | 农民、工匠、冒险者等 | 60% |
| **奴隶** | （部分地区存在） | 1% |
| **超凡者** | 魔法师、战士等修炼者 | 30%（含各级） |

### 精灵社会

| 阶层 | 描述 | 占比 |
|------|------|------|
| **王室** | 精灵王及其家族 | 0.1% |
| **长老** | 各氏族长老 | 1% |

| **平民** | 普通精灵 | 80% |
| **德鲁伊** | 自然守护者 | 9% |

### 兽人社会

| 阶层 | 描述 | 占比 |
|------|------|------|
| **酋长** | 部落首领 | 0.1% |
| **萨满** | 精神领袖 | 2% |

## 日常生活

### 人族日常

| 活动 | 描述 |
|------|------|
| **工作** | 农民种地、工匠制作、商人贸易 |
| **学习** | 富裕家庭送孩子去学院学习 |
| **娱乐** | 酒馆听故事、观看角斗士比赛、节日庆典 |
| **信仰** | 去神庙祈祷，供奉神明 |
| **饮食** | 主食为面包和肉类，富裕家庭有更多选择 |

### 精灵日常

| 活动 | 描述 |
|------|------|
| **修行** | 冥想、学习魔法、练习弓箭 |
| **艺术** | 音乐、绘画、诗歌创作 |
| **自然** | 与植物沟通、照料森林 |
| **社交** | 茶会、音乐会、长老会议 |
| **饮食** | 以水果、花蜜、精灵面包为主 |

### 兽人日常

| 活动 | 描述 |
|------|------|
| **训练** | 战斗训练、体能锻炼 |
| **狩猎** | 狩猎野兽获取食物 |
| **仪式** | 战舞、祭祀、成人礼 |
| **社交** | 篝火晚会、歌谣传唱 |
| **饮食** | 以肉类为主，饮用烈酒 |

## 教育体系

### 魔法教育

| 阶段 | 年龄 | 内容 |
|------|------|------|
| **启蒙** | 6-10岁 | 感知魔力、学习基础理论 |
| **初级** | 10-14岁 | 学习基础魔法、冥想技巧 |
| **中级** | 14-18岁 | 专修某一属性、学习战斗魔法 |
| **高级** | 18-22岁 | 研究高阶魔法、参与实战 |
| **研究** | 22岁+ | 创造新魔法、研究法则 |

### 战士教育

| 阶段 | 年龄 | 内容 |
|------|------|------|
| **基础** | 6-12岁 | 体能训练、基础武器使用 |
| **进阶** | 12-16岁 | 斗气修炼、战技学习 |
| **实战** | 16-20岁 | 参与战斗、积累经验 |
| **精进** | 20岁+ | 突破瓶颈、领悟领域 |

## 法律与秩序

### 人族法律

| 罪行 | 惩罚 |
|------|------|
| **偷窃** | 罚款、监禁、鞭刑 |
| **抢劫** | 监禁、断手 |
| **谋杀** | 死刑 |
| **叛国** | 死刑、家族连坐 |
| **使用禁忌魔法** | 监禁或死刑 |

### 精灵法律

| 罪行 | 惩罚 |
|------|------|
| **破坏自然** | 驱逐、放逐 |
| **伤害同胞** | 监禁、剥夺魔法 |
| **泄露秘密** | 放逐 |
| **背叛王庭** | 死刑 |

### 兽人法律

| 罪行 | 惩罚 |
|------|------|
| **怯懦** | 鞭刑、剥夺战士身份 |
| **背叛部落** | 决斗或死刑 |
| **偷窃** | 鞭刑 |
| **杀害同胞** | 决斗或死刑 |

## 节日与庆典

| 节日 | 时间 | 描述 |
