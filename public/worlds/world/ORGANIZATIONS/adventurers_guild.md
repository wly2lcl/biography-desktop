# ⚔️ 冒险者公会详细设定

> "无论你是谁，来自哪里，只要你敢于冒险，这里就是你的家。"

## 公会概述

冒险者公会是凡人界最大的中立组织，由多种族联合成立，旨在为冒险者提供任务、评级、情报和支援。

### 公会宗旨

- **中立：** 不参与任何政治纷争
- **公正：** 任务评价和奖励公正透明
- **互助：** 冒险者之间互相帮助
- **探索：** 探索未知，发现新事物

## 公会架构

### 总部与分部

| 分部 | 位置 | 描述 |
|------|------|------|
| **总部** | 圣辉城（奥兰帝国帝都） | 最大的冒险者公会，评级中心 |
| **中央分部** | 中央平原各大城市 | 处理日常任务 |
| **西海岸分部** | 自由贸易城邦 | 海洋任务、贸易护送 |
| **翡翠分部** | 翡翠森林边缘 | 精灵领地任务（需精灵许可） |
| **荒原分部** | 龙骨荒原边缘 | 兽人领地任务（需兽人许可） |
| **北境分部** | 极北冰原边缘 | 冰原探索任务 |

| 职位 | 描述 | 人数 |
|------|------|------|

## 任务系统

### 任务发布流程

1. **委托人** 到公会发布任务
2. **任务发布员** 评估任务难度，确定等级和奖励
3. **任务** 张贴在公告板上
4. **冒险者** 接受任务
5. **完成任务** 后返回公会报告
6. **评级官** 验证任务完成情况
7. **发放奖励** 和公会积分

### 任务类型

| 类型 | 描述 | 举例 |
|------|------|------|
| **讨伐** | 消灭指定怪物 | 讨伐狼群、消灭炎魔 |
| **护送** | 保护某人或某物安全到达 | 护送商队、保护学者 |
| **探索** | 探索未知区域或遗迹 | 探索远古遗迹、绘制地图 |
| **采集** | 收集指定材料 | 采集魔法草药、挖掘矿石 |
| **调查** | 调查某事件或现象 | 调查村庄怪事、追踪逃犯 |
| **救援** | 救援被困人员 | 救援失踪冒险者、解救俘虏 |
| **守卫** | 保卫某地点 | 守卫村庄、保护商路 |

### 任务评价

| 评价 | 描述 | 影响 |
|------|------|------|
| **完美** | 超出预期完成 | 额外奖励+50%，公会积分+100% |
| **优秀** | 高质量完成 | 额外奖励+20% |
| **完成** | 按要求完成 | 正常奖励 |
| **勉强** | 勉强完成，有损失 | 奖励-20% |
| **失败** | 未完成 | 无奖励，可能扣分 |

## 公会福利

| 福利 | 描述 | 适用等级 |
|------|------|----------|
| **任务大厅** | 接受各种任务 | 所有等级 |
| **情报服务** | 获取怪物、遗迹情报 | 银牌及以上 |
| **医疗援助** | 免费基础治疗 | 所有等级 |
| **装备租赁** | 租赁公会装备 | 金牌及以上 |
| **训练场** | 使用公会训练设施 | 银牌及以上 |
| **传送服务** | 使用公会传送阵 | 白金及以上 |
| **紧急救援** | 遇到危险时呼叫救援 | 钻石及以上 |
| **特殊任务** | 接受S/SS级任务 | 传奇 |

## 公会规则

| 规则 | 描述 | 违反后果 |
|------|------|----------|
| **不得伤害同伴** | 禁止攻击其他冒险者 | 降级或开除 |
| **不得接受邪恶任务** | 禁止参与邪恶活动 | 开除并通缉 |
| **不得隐瞒任务信息** | 必须如实报告任务情况 | 警告或扣分 |
| **不得破坏公会设施** | 爱护公会财产 | 赔偿并警告 |
| **尊重所有种族** | 禁止种族歧视 | 警告或降级 |
