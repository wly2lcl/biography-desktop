# 🗣️ 语言与文字

> 语言是文明的基石，不同的种族有着不同的交流方式。

## 主要语言

### 通用语

| 属性 | 描述 |
|------|------|
| **名称** | 通用语（Common） |
| **使用种族** | 所有种族 |
| **起源** | 由人族和精灵族共同发展 |
| **特点** | 语法简单，易于学习 |
| **文字** | 拉丁字母变体 |

### 精灵语

| 属性 | 描述 |
|------|------|
| **名称** | 精灵语（Elvish） |
| **使用种族** | 精灵族 |
| **特点** | 优美如歌，大量使用元音 |
| **文字** | 精灵符文，形似树叶和花朵 |
| **特殊** | 部分精灵语词汇本身具有魔力 |

### 兽人语

| 属性 | 描述 |
|------|------|
| **名称** | 兽人语（Orcish） |
| **使用种族** | 兽人族 |
| **特点** | 粗犷直接，大量使用喉音 |
| **文字** | 无文字，口述传承 |
| **特殊** | 战吼是兽人语的重要部分 |

### 龙语

| 属性 | 描述 |
|------|------|
| **名称** | 龙语（Draconic） |
| **使用种族** | 龙族 |
| **特点** | 世间最古老的语言，每个音节都蕴含力量 |
| **文字** | 龙语符文，形似龙鳞 |
| **特殊** | 使用龙语施放魔法威力远超常规魔法 |

### 矮人语

| 属性 | 描述 |
|------|------|
| **名称** | 矮人语（Dwarvish） |
| **使用种族** | 矮人族 |
| **特点** | 硬朗有力，大量使用辅音 |
| **文字** | 矮人符文，刻在石头和金属上 |
| **特殊** | 锻造时必须用矮人语念诵咒文 |

### 魔族语

| 属性 | 描述 |
|------|------|
| **名称** | 魔族语（Demonic） |
| **使用种族** | 魔族 |
| **特点** | 低沉神秘，带有回音效果 |
| **文字** | 魔族符文，形似火焰和暗影 |
| **特殊** | 契约必须用魔族语书写才具有魔法约束力 |

## 语言学习

### 学习难度

| 语言 | 学习难度 | 描述 |
|------|----------|------|
| 通用语 | ★ | 最简单，大部分人都懂 |
| 精灵语 | ★★★ | 需要良好的音感 |
| 兽人语 | ★★ | 语法简单，但发音困难 |
| 龙语 | ★★★★★ | 极难学习，非龙族几乎无法掌握 |
| 矮人语 | ★★★ | 需要理解锻造文化 |
| 魔族语 | ★★★★ | 需要理解契约和暗影文化 |

### 语言天赋

| 种族 | 语言天赋 | 描述 |
|------|----------|------|
| 人族 | ★★★★★ | 学习语言最快 |
| 精灵 | ★★★★ | 音感好，擅长学习优美语言 |
| 兽人 | ★★ | 不擅长学习其他语言 |
| 龙族 | ★★★ | 懂所有语言，但很少使用 |
| 矮人 | ★★★ | 擅长学习硬朗的语言 |
| 魔族 | ★★★★ | 擅长理解契约语言 |

## 文字与魔法

### 魔法文字

某些文字本身具有魔力：

| 文字 | 魔力 | 用途 |
|------|------|------|
| **精灵符文** | 自然魔法 | 制作魔法卷轴、结界 |
| **龙语符文** | 龙语魔法 | 龙语魔法施法 |
| **矮人符文** | 锻造魔法 | 附魔、锻造 |
| **魔族符文** | 暗影魔法 | 契约、诅咒 |
| **神文** | 神术 | 神术施法、神器制造 |

### 古代文字

| 文字 | 描述 | 现状 |
|------|------|------|
| **创世文字** | 创世神使用的文字 | 已失传，仅存于传说 |
| **远古文字** | 上古文明使用的文字 | 部分遗迹中可见 |
| **神战文字** | 神战时期使用的文字 | 学者正在研究中 |
