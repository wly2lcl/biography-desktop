# 📝 角色创建指南

> 每个角色都是世界的一部分，他们的故事将推动历史前进。

## 角色创建步骤

### 一、选择种族

| 种族 | 适合职业 | 特色 |
|------|----------|------|
| **人族** | 所有职业 | 适应性强，成长快 |
| **精灵** | 魔法师、德鲁伊 | 寿命长，魔法亲和 |
| **兽人** | 战士、萨满 | 力量强大，狂暴天赋 |
| **矮人** | 战士、锻造师 | 锻造大师，毒素免疫 |
| **半身人** | 盗贼、游侠 | 好运，隐匿 |
| **魔族** | 暗系魔法师、契约师 | 黑暗魔法天赋 |
| **海族** | 水系魔法师、战士 | 水下战斗优势 |
| **龙族** | 龙语魔法师、战士 | 极其强大，但成长慢 |

### 二、选择职业

| 职业 | 核心属性 | 战斗风格 | 成长曲线 |
|------|----------|----------|----------|
| **魔法师** | 智力、精神力 | 远程、范围伤害 | 前期弱，后期强 |
| **战士** | 力量、体质 | 近战、持续输出 | 稳定成长 |
| **刺客** | 敏捷、暴击 | 单体爆发 | 前期强，后期依赖技巧 |
| **牧师** | 信仰、精神力 | 辅助、治愈 | 团队核心 |
| **炼金术师** | 智力、感知 | 辅助、制造 | 非战斗，但极有价值 |

### 三、确定背景

| 背景 | 描述 | 起始优势 |
|------|------|----------|
| **贵族** | 出生在贵族家庭 | 资金、教育、人脉 |
| **平民** | 普通家庭出身 | 坚韧、接地气 |
| **孤儿** | 无父母，独自成长 | 独立、生存技能 |
| **学徒** | 在某位强者门下学习 | 基础技能、导师资源 |
| **冒险者** | 已有冒险经验 | 实战经验、装备 |
| **学者** | 研究魔法或历史 | 知识、研究能力 |
| **军人** | 曾在军队服役 | 战斗训练、纪律 |
| **流浪者** | 四处游历 | 见识广、适应力强 |

### 四、设定性格

| 性格维度 | 选项 |
|----------|------|
| **外向/内向** | 喜欢社交 vs 喜欢独处 |
| **乐观/悲观** | 看到希望 vs 看到危险 |
| **理性/感性** | 逻辑思考 vs 情感驱动 |
| **勇敢/谨慎** | 直面危险 vs 谨慎行事 |
| **忠诚/自私** | 为团队着想 vs 为自己着想 |

### 五、选择动机

| 动机 | 描述 |
|------|------|
| **复仇** | 为家人或朋友复仇 |
| **权力** | 追求更高的地位和力量 |
| **知识** | 探索未知的奥秘 |
| **财富** | 追求金钱和物质 |
| **保护** | 保护某人或某地 |
| **救赎** | 弥补过去的错误 |
| **自由** | 追求无拘无束的生活 |
| **荣誉** | 追求名声和尊重 |

## 角色模板

### 模板一：天才魔法师

| 属性 | 描述 |
|------|------|
| **种族** | 人族或精灵 |
| **职业** | 魔法师 |
| **背景** | 贵族或学徒 |
| **性格** | 理性、好奇、略显傲慢 |
| **动机** | 探索魔法的终极奥秘 |
| **天赋** | 极高的魔力亲和和悟性 |
| **弱点** | 身体脆弱，缺乏实战经验 |
| **成长线** | 从学院天才到触摸法则的强者 |

### 模板二：兽人战士

| 属性 | 描述 |
|------|------|
| **种族** | 兽人 |
| **职业** | 战士（狂战士） |
| **背景** | 部落战士 |
| **性格** | 勇敢、直率、重视荣誉 |
| **动机** | 证明自己是部落最强者 |
| **天赋** | 狂暴天赋，肉体恢复力强 |
| **弱点** | 不擅长魔法，智力较低 |
| **成长线** | 从部落战士到战神境界 |

### 模板三：刺客游侠

| 属性 | 描述 |
|------|------|
| **种族** | 半身人或精灵 |
| **职业** | 刺客 |
| **背景** | 流浪者或孤儿 |
| **性格** | 谨慎、独立、略带悲观 |
| **动机** | 寻找失踪的亲人 |
| **天赋** | 好运或隐匿能力 |
| **弱点** | 正面战斗能力弱 |
| **成长线** | 从街头小偷到暗影大师 |

### 模板四：混血冒险者

| 属性 | 描述 |
|------|------|
| **种族** | 人族/精灵混血 或 人族/兽人混血 |
| **职业** | 战士+魔法师（双修） |
| **背景** | 孤儿或学徒 |
| **性格** | 坚韧、适应力强、略带自卑 |
| **动机** | 证明混血不比纯血差 |
| **天赋** | 兼容并蓄，可双修 |
| **弱点** | 两个体系都不精通 |
| **成长线** | 从被歧视的混血到开创全新体系 |

## 角色成长曲线

### 魔法师成长

| 阶段 | 年龄 | 实力 | 主要活动 |
|------|------|------|----------|
| **学院期** | 10-18岁 | 见习~初级 | 学习基础魔法 |
| **成长期** | 18-25岁 | 中级~高级 | 冒险者任务，实战经验 |
| **成熟期** | 25-40岁 | 大魔法师~魔导师 | 研究魔法，成为一方强者 |
| **巅峰期** | 40-100岁 | 大魔导师~圣域 | 探索法则，创造新魔法 |
| **传说期** | 100岁+ | 半神~神明 | 影响世界格局 |

### 战士成长

| 阶段 | 年龄 | 实力 | 主要活动 |
|------|------|------|----------|
| **训练期** | 10-16岁 | 见习~初级 | 基础训练 |
| **成长期** | 16-25岁 | 中级~高级 | 军队服役或冒险 |
| **成熟期** | 25-40岁 | 大战士~战将 | 成为将领或强者 |
| **巅峰期** | 40-100岁 | 战神~圣域 | 领悟领域 |
| **传说期** | 100岁+ | 半神~神明 | 影响世界格局 |

## 角色关系网

每个角色都应该有以下关系：

| 关系类型 | 描述 | 举例 |
|----------|------|------|
| **导师** | 教导角色的人 | 学院导师、军队教官 |
| **朋友** | 与角色并肩作战的人 | 队友、同学 |
| **敌人** | 与角色对立的人 | 竞争对手、仇人 |
| **爱人** | 角色的情感寄托 | 恋人、配偶 |
| **家人** | 角色的血缘关系 | 父母、兄弟姐妹 |
| **恩人** | 对角色有恩的人 | 救命恩人、资助者 |
