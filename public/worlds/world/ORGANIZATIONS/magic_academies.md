## 学院体系

### 学制

| 阶段 | 时长 | 内容 |
|------|------|------|
| **预科** | 1-2年 | 魔力感知、基础理论、通用魔法知识 |
| **初级** | 2-3年 | 基础魔法练习、冥想技巧、属性选择 |
| **中级** | 3-4年 | 高级魔法、战斗技巧、魔法理论 |
| **高级** | 4-5年 | 研究课题、魔法创造、法则感悟 |
| **研究** | 不限 | 原创研究、发表论文、探索未知 |

### 学院生活

| 活动 | 描述 |
|------|------|
| **课程** | 理论课、实践课、冥想课 |
| **考试** | 每学期一次，包括理论和实践 |
| **竞赛** | 学院间的魔法竞赛，展示学生实力 |
| **实习** | 高年级学生参与冒险者公会任务 |
| **研究** | 高年级学生进行原创魔法研究 |

### 学院资源

| 资源 | 描述 |
|------|------|
| **图书馆** | 收藏大量魔法书籍和古籍 |
| **实验室** | 进行魔法实验和研究 |
| **训练场** | 练习魔法和战斗 |
| **冥想室** | 安静的冥想环境 |
| **魔法阵** | 辅助修炼的特殊设施 |
