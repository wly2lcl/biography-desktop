# 📖 神话与传说

> 在历史的长河中，有些故事已经无法分辨真假。

## 创世神话

### 阿鲁恩的创世

远古之前，只有混沌和虚无。**创世神阿鲁恩**从混沌中苏醒，他感到孤独，于是用自身神力创造了世界。

他用七种情绪创造了七个种族：
- **喜悦** → 精灵
- **愤怒** → 兽人
- **好奇** → 人族
- **坚忍** → 矮人
- **温柔** → 半身人
- **威严** → 龙族
- **忧虑** → 魔族

创造完成后，阿鲁恩陷入沉睡。据说他会在世界面临最大危机时再次苏醒。

### 众神的诞生

阿鲁恩沉睡后，他的神力碎片化为第一批神明：

| 神明 | 领域 | 描述 |
|------|------|------|
| **生命女神·艾莉娅** | 生命、治愈 | 最温柔的神，创造了海族 |
| **战神·马尔斯** | 战争、勇气 | 最狂暴的神，深受兽人崇拜 |
| **智慧之神·索伦** | 知识、魔法 | 最睿智的神，魔法的创造者 |
| **死神·哈迪斯** | 死亡、灵魂 | 最神秘的神，在神战中陨落 |
| **秩序之神·朱庇特** | 秩序、法律 | 最公正的神，维持世间秩序 |
| **混沌之主·卡俄斯** | 混沌、毁灭 | 最危险的存在，被封印于深渊 |

## 著名传说

### 雷欧·铁拳的故事

人族战士雷欧·铁拳，出身平民，却凭借惊人的毅力从见习战士一路修炼至战神境界。

传说他曾在年轻时单挑一头远古红龙，虽然惨败，但红龙被他的勇气打动，没有杀他，反而传授了龙语魔法的基础。

后来，雷欧成为第一位同时掌握战士体系和魔法体系的人类，被誉为"战神法王"。

### 永恒塔的传说

永恒塔是一座悬浮于空中的神秘高塔，不属于任何势力。据说塔内有无穷的知识，但只有被塔认可的人才能进入。

永恒塔每隔百年会开放一次，吸引世界各地的冒险者和学者前来探索。但进入塔中的人，有些获得了无上的知识，有些却再也没有出来。

关于永恒塔的来历，有几种说法：
- 创世神阿鲁恩留下的遗产
- 某位远古神明的居所
- 一个独立的位面，只是外观像塔
- 某位超越神明的存在的创造物

### 龙之墓地的诅咒

远古时代，一场巨龙之间的内战导致无数巨龙死亡。幸存的巨龙将同伴的遗骸集中埋葬，形成了龙之墓地。

传说墓地中蕴含着恐怖的龙威，任何进入的生物都会感受到来自灵魂深处的恐惧。但同时，墓地中也散落着巨龙的遗骨和鳞片，是锻造神器的最佳材料。

据说，只有被龙族认可的生物，才能安全进入龙之墓地。
