## 剧情模板

### 模板一：学院成长

```
起点：主角进入魔法学院
发展：学习魔法、结交朋友、参加竞赛
转折：发现学院隐藏的秘密
高潮：学院面临危机，主角挺身而出
结局：主角成长为强者，揭开秘密
```

### 模板二：复仇之路

```
起点：主角的家人/朋友被杀
发展：追查凶手，提升实力
转折：发现凶手背后有更大阴谋
高潮：与最终BOSS对决
结局：复仇成功，但代价沉重
```

### 模板三：种族冲突

```
起点：主角卷入种族冲突
发展：了解冲突根源，寻找和平方案
转折：发现有人在幕后操纵
高潮：阻止战争，揭露阴谋
结局：种族和解，主角成为英雄
```

### 模板四：探索未知

```
起点：主角发现远古遗迹的线索
发展：组建探险队，进入遗迹
转折：发现遗迹中的危险和秘密
高潮：面对遗迹守护者
结局：获得宝藏/知识，但付出代价
```
